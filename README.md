接下来需要处理余下other部分，目前分析情况是”分句“类处理不当，会考虑添加这条判断之后再进行处理

目前模型的想法有两个：

1、 一个RNN去除冗余属性，之后导入规则，输出结果

2、 RNN去除冗余属性之后，再进行分类模型训练

第一种准确率需要两方都很高，需要细化RNN模型，完善规则（目前正在做的）

第二种容易出现冗余操作，多次训练的效率可能不高，但是提升准确率的可能性较高
